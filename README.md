# Data Analysis Project

##  Overview
This project analyzes sales data using Python (Pandas, Matplotlib, Seaborn).

##  Steps
1. Data Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Visualization  

##  Libraries Used
- pandas  
- numpy  
- matplotlib  
- seaborn  

##  Results
The analysis shows key sales trends and relationships between variables.
